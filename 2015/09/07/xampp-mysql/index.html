MacOS中XAMPP MySQL无法启动

By Jack.LN

2015-09-07 更新日期:2015-09-07

文章目录
  1. 1. 在终端中自己开启
  2. 2. 设置权限

在终端中自己开启

sudo /Applications/XAMPP/xamppfiles/bin/mysql.server start

每次都要这样开启

设置权限

sudo emacs /Library/LaunchDaemons/com.mysql.mysql.plist

sudo launchctl load -w /Library/LaunchDaemons/com.mysql.mysql.plist

这样每次就都可以使用xampp的控制台来启动了
