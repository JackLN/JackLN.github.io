Kali安装4.0内核

使用新版的Kali想装驱动想自己做一些工具，需要快速的搞一个linux内核！

安装依赖库

一些内核的编译，安装所依赖的库

apt-get install kernel-package ncurses-dev fakeroot bzip2

下载内核

下载Kali Linux内核源码

apt-get install linux-source
cd /usr/src/
tar xvJf linux-source-4.0.tar.xz
cd linux-source-4.0/

配置内核

cp /boot/config-4.0.0-kali1-amd64 .config
make menuconfig

这里可能出错…主要是前面依赖库没有装好。还有就是可能你和我一样眼神不好使用的大字体小窗口，放大窗口即可。直接Save就可以了。

编译内核

CONCURRENCY_LEVEL=$(cat /proc/cpuinfo|grep processor|wc -l)
make-kpkg clean
fakeroot make-kpkg kernel_image

这一步可能要花上小半天时间，机子好点可能会快一点。所以找个适当时间来编译。

安装内核

dpkg -i ../linux-image-4.0.4_4.0.4-10.00.Custom_amd64.deb
update-initramfs -c -k 4.0.4
update-grub2
reboot

享受内核编程吧！哈哈哈哈
真正的来去自如，却是真正的来去空空啊！ 啊哈哈哈！
