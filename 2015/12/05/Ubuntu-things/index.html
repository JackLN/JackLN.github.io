Ubutun安装NVIDIA显卡驱动

如果你的Ubuntu在lightdm中鼠标总会闪烁，你就要装显卡驱动了。。。
如果你想在你的Ubuntu上玩dota2，你就要装显卡驱动了。。。
我发现的方法有两种：
a. 在NVIDIA官网上搜索自己显卡的型号，然后现在对应的驱动进行安装。
在安装新的驱动前需要卸载之前的NVIDIA驱动，并且准确的删除残留文件。
如果是新的干净的Ubuntu系统推荐使用此方法安装驱动。
b. 进入设置－> software&update－>Additional Drivers。
选择需要安装的驱动即可。安装重启后不稳定，会出现奇奇怪怪的问题。
不过遇到的问题都可以在网上找到解决的方法。台式机或不同的笔记本型号
会出现不同的问题，就不一一列举了。（最直接的方法是在BIOS中禁用集显）
不要装过于新版本的驱动。
最后装入一个管理工具 NVIDIA X server settings
